#App #Config => Pasta com os defines de configuração´do Lidiun.
#App #Api => Pasta para APIs como a do Facebook e PagSeguro.

#App #Repository #Layout #Segment => Arquivos html com os segmentos de html usados em qualquer render.
#App #Repository #Render => Arquivos php com as classes que controlam as páginas requeridas.
